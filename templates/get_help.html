/query

This is an interface allowing direct links to the dep_search web user interface.

  • db : a tree database to query (see the currently available list below). Compulsory parameter.
  • search : the query to run
  • case : case-sensitive "true" or "false"
  • max_hits : how many hits to show? 10, 50, 100, 250 are possible values
